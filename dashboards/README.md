# Dashboards

This folder stores Home Assistant dashboard YAML and related dashboard exports.

- `overview.yaml` is the exported Overview dashboard registered as `overview-dashboard` in `configuration.yaml`.
- `hoefijzer.yaml` is the Hoefijzer engineering dashboard registered as `hoefijzer-dashboard` in `configuration.yaml`.
- `energy_diagnostics.json` is the Home Assistant energy dashboard diagnostics export.
